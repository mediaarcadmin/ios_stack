Book Place includes, in statically linked library form, a heavily modified version of Steve Wolter's libhyphenate library, used for text hyphenation. It is distributed under the GNU Lesser Public License version 2.1, with an explicit exception granted by Steve Wolter allowing static linking "without the need for the application/library linked to be published under LPGL".

Book Place includes, in statically linked library form, the expat xml parser. It is distributed under the MIT License.

Book Place includes, in statically linked library form, the VCTitleCase library for converting text to title case. It is distributed under the MIT License.

Book Place uses fonts in the LinuxLibertine font family. The fonts are distributed under the SIL Open Font License, Version 1.0.

Book Place includes, in statically linked library form, the libwapcaplet library for string internment. It is distributed under the MIT License.

Book Place includes, in statically linked library form, the libparserutils library for building efficient parsers. It is distributed under the MIT License.

Book Place includes, in statically linked library form, the libcss library for CSS parsing. It is distributed under the MIT License.

Blio includes, in statically linked library form, a heavily modified version of Steve Wolter's libhyphenate library, used for text hyphenation. It is distributed under the GNU Lesser Public License, version 2.1 with an explicit exception granted by Steve Wolter allowing static linking "without the need for the application/library linked to be published under LPGL".

Blio includes, in statically linked library form, the expat xml parser. It is distributed under the MIT License.

Blio includes, in statically linked library form, the VCTitleCase library for converting text to title case. It is distributed under the MIT License.

Blio uses fonts in the LinuxLibertine font family. The fonts are distributed under the SIL Open Font License, Version 1.0.

Blio includes, in statically linked library form, the libwapcaplet library for string internment. It is distributed under the MIT License.

Blio includes, in statically linked library form, the libparserutils library for building efficient parsers. It is distributed under the MIT License.

Blio includes, in statically linked library form, the libcss library for CSS parsing. It is distributed under the MIT License.

Blio includes the ZipArchive class for zip/unzip. It is distributed under the MIT License.

Blio includes, in statically linked library form, the GData library for reading web data. It is distributed under the Apache License, version 2.0.
